Lab 19: AJAX

Challenge

The challenge of this assignment was to use an API to generate data when clicking a button. To accomplish this, I used an API that generates random Kanye West quotes. Hope you enjoy.

Problems

My first challenge was understanding Task 2. My partner and I were confused, so I asked my friend Jayson and he explained that docs refered to the API's documentation. I had a lot of trouble trying to find an API that wasn't too complicated and functioned. I tried to understand various API's which included an API that generates random color palettes for designers to use. Then I stumbled upon kanye.rest, which is an API that generates random Kanye West quotes. One of my challenges for my code, was that when I tried using the html() method, after clicking the button, the code overrided my own css and JavaScript and printed only the quote to my page. I broke my code a few times by making minor changes to my code and I tried checking in my console for errors, but none of that helped. I lastly reached out to Wes during his office hours, and it turned out I had the output div around the content. I got my code to work after all.
